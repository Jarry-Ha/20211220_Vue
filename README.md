## Vue 설치 및 프로젝트 생성
1. 설치  
npm info @vue/cli<br>
npm i -g @vue/cli

   버전 확인  
   vue --version
   <br /><br />

2. 프로젝트 생성<br>
vue create projectName 

   프로젝트 시작  
   cd projectName<br>
   npm run serve
   <br /><br />
